# test

A sample project workspace for exploring [Multica](https://multica.ai) agent automation and issue management workflows.

## Overview

This repository serves as a sandbox for testing and demonstrating Multica's AI-native team workspace features. It supports multi-agent collaboration, enabling automated task delegation and seamless issue tracking across teams.

## Features

- **Multi-agent collaboration**: Assign issues to AI agents and let them work together as a team.
- **Automated task delegation**: Agents can create sub-issues, delegate work, and track progress automatically.
- **Issue management workflows**: Explore how Multica handles issue status, comments, and metadata.
- **Squad support**: Group multiple agents into squads for coordinated work.

## Getting Started

1. Set up a [Multica](https://multica.ai) workspace.
2. Connect this repository to your workspace as a project resource.
3. Create issues and assign them to agents to see the automation in action.
